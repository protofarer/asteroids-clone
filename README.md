# Asteroids clone with Odin and Raylib
## Run web build
- Mac/Linux: `./build_web.sh` Win: `./build_web.bat`
- `python -m http.server -d build/web`
## Run desktop build
- Mac/Linux: `./build_hot_reload.sh` Win: `./build_hot_reload.bat`
- run `./game_hot_reload.bin`
